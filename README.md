# Smart Library Management System (SLMS)

This project is a console-based Smart Library Management System developed in C++ using Object-Oriented Programming.

## Features
- Member account creation
- Member login
- View book catalogue
- Borrow books
- Return books
- Reserve unavailable books
- Borrow limit of 5 books
- Admin login
- View overdue books
- Advance system time
- Generate system report
- Reset system data

## OOP Concepts Used

### Inheritance
Member and Admin classes inherit from the User base class.

### Encapsulation
Classes such as Library manage their own data and rules.

### Polymorphism
Different user roles implement the getRole() function differently.

## How to Run

1. Open the project in Visual Studio.
2. Build the solution.
3. Run the program.
